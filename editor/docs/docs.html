About Bitsy advanced dialog & scripting

Using the advanced dialog panel or scripting is a way to make your game's dialog more dynamic. It allows the text a character says to change each time you talk to them, or based on what items the player has, or other variables. It also allows you add special effects to your text such as rainbow text or shaky text. Most advanced dialog features available through the UI or through scripting, but some are only available throught scripting (so far).
